# reg-expression

🚀 reg-expression 🌈.

## usage
### 安装
```bash
npm i reg-expression --save
```

### 使用
```javascript
import RegExpression from 'reg-expression'
// or
import { ip_v4 } from 'reg-expression'

const isIPV4 = ip_v4.test('1.1.1.1')
console.log(isIPV4) // true
```
